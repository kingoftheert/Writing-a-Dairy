# Writing-a-Dairy

A website tool to save your dairy by an easy cipher.

"abcdefghijklmnopqrstuvwxyz0123456789"

The keyword is make of "the first three letters the days of the week" + "the last two letters of the day" + "the first three letters of the month" + "year"

EX: "tuendapr2025"
